Laboratorium Teknik Informatika & Sistem Informasi

Fasilitas pendukung pembelajaran praktis dan inovasi teknologi

Fasilitas Laboratorium

Laboratorium Pemrograman

Laboratorium ini digunakan untuk praktikum pemrograman dasar, pemrograman lanjut, dan pengembangan perangkat lunak. Dilengkapi dengan komputer berbasis Windows dan Linux, serta akses internet cepat untuk mendukung pembelajaran mahasiswa.

  • PC berbasis Windows dan Linux
  • Internet 100 Mbps
  • Buka setiap hari kerja

Laboratorium Jaringan

Laboratorium jaringan komputer menyediakan fasilitas untuk belajar tentang sistem jaringan, keamanan jaringan, dan pengembangan infrastruktur digital. Mahasiswa dapat melakukan simulasi jaringan dan praktikum keamanan data di sini.

  • Perangkat jaringan (router, switch, dll.)
  • Praktikum keamanan jaringan
  • Simulasi topologi jaringan

Laboratorium AI & Data Science

Laboratorium ini dirancang untuk pengembangan sistem berbasis kecerdasan buatan dan analisis data besar. Mahasiswa dapat mengakses perangkat keras dengan spesifikasi tinggi dan tools seperti Python, TensorFlow, dan Hadoop.

  • Pengembangan sistem AI
  • Analisis data dengan Python
  • Komputer dengan GPU

Laboratorium UI/UX & Mobile Development

Laboratorium ini digunakan untuk praktikum desain antarmuka pengguna (UI), pengalaman pengguna (UX), dan pengembangan aplikasi mobile Android & iOS. Mahasiswa dilatih untuk menciptakan aplikasi yang intuitif dan user-friendly.

  • Praktikum mobile development
  • Tools desain UI/UX (Figma, Adobe XD)
  • Prototyping aplikasi

Laboratorium Teknologi Cloud

Laboratorium ini dirancang untuk mendukung pembelajaran komputasi awan, cloud architecture, dan serverless computing. Mahasiswa dapat menggunakan platform seperti AWS, Google Cloud, dan Microsoft Azure untuk membangun infrastruktur digital berbasis cloud.

  • Akses ke AWS & Google Cloud
  • Simulasi cloud infrastructure
  • Praktikum serverless computing
